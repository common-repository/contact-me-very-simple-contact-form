== Description ==

Contact Me is a very simple contact form with an uploading file option.

SHORTCODE: [vv-contact-form]

The plugin is created on WordPress 4.9 and it is not recomended to use it on previous version of WordPress.

If you want to use it, at first you need to set up your mail setting in WordPress Setting or use relevant plugin for that. The plugin sends you emails, so setting up your email is mandatory.

The plugin generate a contact form, when you use its shortcode: [vv-contact-form]. 
The form consist Name Field, Email Field, Subject Field, Message Field and optionally Upload File Field; When the site owner allow the upload field, uploading a file is required. 

There are some validations:
Name - > Only letters, Maximum 50 characters, Required (the user cant leave it blank)
Email - > Standart Email Validation, Maximum 50 characters, Required (the user cant leave it blank)
Subject - > Maximum 50 characters, Required (the user cant leave it blank), Required (the user cant leave it blank)
Message - > Maximum 1000 characters, Required (the user cant leave it blank)
Upload File - > Maximum upload size -2MB, Allowed formats: "pdf", "doc", "docx", "gif", "jpeg", "jpg", "png", "txt", Required (the user cant leave it blank)

Enjoy :)


== Installation ==



1. Upload the plugin files to the `/wp-content/plugins` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Use the Settings->Contact Me screen to configure the upload option.


== Frequently Asked Questions ==

= The form dont send me email after submit. =

Make sure that you have set up your email settings.

= There are some errors when I submit the form =

The form fields have validation. Make sure you enter valid data.
